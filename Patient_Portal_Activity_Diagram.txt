# MEDIKOS PATIENT PORTAL - UML ACTIVITY DIAGRAM FLOW

## MAIN PATIENT JOURNEY
START → Login Check → Dashboard → Feature Selection → Feature Workflow → END

## 1. AUTHENTICATION FLOW
User Opens App → Login Required? → [YES: Enter Credentials → Validate → Dashboard] / [NO: Direct to Dashboard]

## 2. DASHBOARD OVERVIEW
Load Health Metrics → Display Quick Actions → User Selects Feature

## 3. DISEASE PREDICTION WORKFLOW
Select Symptoms → Validate Input → AI Analysis → Display Results → Save/Retry/Exit

## 4. GENERIC MEDICINE WORKFLOW  
Enter Medicine Name → Search Database → Show Alternatives → Compare Prices → Save/Search Again/Exit

## 5. HEALTH DIARY WORKFLOW
[New Entry: Write Content → Analyze Text → Save Entry] / [View Entries: Browse Calendar → Select Entry] / [Analytics: Show Trends]

## 6. AYURVEDIC REMEDIES WORKFLOW
Enter Symptoms → Generate Remedies → [Remedies Tab / Diet Tab] → Display Recommendations → Save/Retry/Exit

## 7. MEDIVAULT WORKFLOW
[Upload: Select File → Fill Details → Process Upload → Save Document] / [Manage: Browse Documents → View/Edit/Delete/Share]

## 8. AI CHATBOT WORKFLOW
Open Chat → Type Message → AI Processing → Display Response → Continue Chat/Save/Exit

## 9. AI VOICE AGENT WORKFLOW
Request Mic Permission → Start Listening → Speech-to-Text → Process Query → Text-to-Speech Response → Continue/Exit

## 10. HEALTH PASSPORT WORKFLOW
Display Health Profile → [Download PDF / Share / Generate QR / Update Info] → Process Action → Complete

## DECISION POINTS
- Authentication Status
- Feature Selection  
- Input Validation
- File Type/Size Check
- Permission Grants
- Continue/Exit Choice

## PARALLEL PROCESSES
- Loading Animations
- Background Data Processing
- Auto-save Functions
- Real-time Updates
- Error Monitoring

## ERROR HANDLING
Network Error → Show Offline Mode
Session Expired → Redirect to Login  
Invalid Input → Show Validation Errors
Service Unavailable → Show Retry Options

## START STATES: App Launch, Feature Click, User Input
## END STATES: Task Complete, Navigation Away, Logout, Error Recovery
